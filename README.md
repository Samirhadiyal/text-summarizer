# 📖 Overview

This project is an end to end AI powered dialogue summarization system that converts lengthy conversations into concise and meaningful summaries using a fine tuned **T5 Small Transformer model**.

Unlike traditional extractive summarizers, this project performs **abstractive summarization**, generating natural language summaries that preserve the core meaning instead of simply copying sentences.

The application includes:

* 🤖 Custom fine tuned Transformer model
* ⚡ FastAPI backend
* 🌐 Responsive web interface
* 🔥 REST API
* 🧠 GPU acceleration support
* 📄 Dialogue summarization using Natural Language Processing
